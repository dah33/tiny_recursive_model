# Tiny Recursive Model (TRM)

Reimplementation using Pytorch Lightning of the [Tiny Recursive Model](https://github.com/SamsungSAILMontreal/TinyRecursiveModels) in the paper, [Less is More: Recursive Reasoning with Tiny Networks](https://arxiv.org/abs/2510.04871) by Alexia Jolicoeur-Martineau.
